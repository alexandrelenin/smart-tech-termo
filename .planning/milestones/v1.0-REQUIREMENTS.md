# Requirements: smart-tech-termo — v1.0 (SHIPPED)

**Defined:** 2026-04-22
**Shipped:** 2026-05-07
**Core Value:** Usuário autenticado cria, salva e imprime termos de entrega de licença de forma rápida e segura.

## v1.0 Requirements — Status Final

### Backend

- [x] **BACK-01**: Sistema tem servidor Hono+Node.js respondendo na porta 3001 — *validated (Fase 1)*
- [x] **BACK-02**: Usuário autenticado pode criar ou atualizar um termo (upsert por ID) — *validated (Fase 3)*
- [x] **BACK-03**: Usuário autenticado pode listar todos os seus termos — *validated (Fase 3)*
- [x] **BACK-04**: Usuário autenticado pode buscar um termo específico por ID — *validated (Fase 3)*
- [x] **BACK-05**: Usuário autenticado pode deletar um termo seu — *validated (Fase 3)*

### Autenticação

- [x] **AUTH-01**: Usuário pode criar conta com email e senha — *validated (Fase 2)*
- [x] **AUTH-02**: Usuário pode fazer login com conta Google (OAuth) — *validated (Fase 2 + UAT Fase 7)*
- [x] **AUTH-03**: Sessão do usuário persiste entre recarregamentos do browser — *validated (Fase 2)*
- [x] **AUTH-04**: Toda rota de termos filtra por `ownerId === session.user.id` — *validated (Fase 3)*

### Gemini (Proxy)

- [x] **GEM-01**: Chave `GEMINI_API_KEY` existe apenas como variável de ambiente do servidor — *validated (Fase 4)*
- [x] **GEM-02**: Usuário pode melhorar descrição de texto via `POST /api/gemini/improve` — *validated (Fase 4)*
- [x] **GEM-03**: Usuário pode gerar conclusão/resumo via `POST /api/gemini/summary` — *validated (Fase 4)*

### Deploy

- [x] **DEPL-01**: App roda em Docker Compose com serviços: postgres, api, nginx — *validated (Fase 6)*
- [x] **DEPL-02**: App está acessível na porta 8095 via nginx — *validated (Fase 6, deploy desklime)*

### UX (não declarados originalmente, adicionados na Fase 7)

- [x] **UX-01**: Tela de login com contraste adequado em dark theme — *validated (Fase 7, UAT pass)*
- [x] **UX-02**: Botão Google OAuth com branding correto (logo "G" 4-cores) — *validated (Fase 7, UAT pass)*
- [x] **UX-03**: Botão "Novo Termo" limpa formulário e navega para Protocolo — *validated (Fase 7, UAT pass após fix emptyData)*
