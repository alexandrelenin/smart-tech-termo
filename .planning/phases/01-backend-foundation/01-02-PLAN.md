Criar `server/index.ts` com o seguinte conteúdo:

```typescript
// server/index.ts
import 'dotenv/config'
import { Hono } from 'hono'
import { serve } from '@hono/node-server'

const app = new Hono()

app.get('/api/health', (c) => {
  return c.json({ status: 'ok', timestamp: new Date().toISOString() })
})

const PORT = Number(process.env.PORT ?? 3001)

serve({ fetch: app.fetch, port: PORT }, (info) => {
  console.log(`Server running on http://localhost:${info.port}`)
})

export default app
```

Notas:
- `import 'dotenv/config'` deve ser a primeira linha para carregar o .env antes de qualquer outro import
- A porta padrão é 3001 (per CLAUDE.md)
- A rota `/api/health` retorna JSON `{ status: 'ok', timestamp: <ISO string> }` com status 200
- O servidor NÃO deve logar DATABASE_URL, BETTER_AUTH_SECRET ou qualquer segredo no console
